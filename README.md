# Python TO-DO-LIST Command line based 📋

A simple To-Do List widely used tool for organizing tasks and managing daily activities. It helps users keep track of what needs to be done, acheive goals, and improve productivity. This project implements a <i> <b> command-line based </b> </i> To-Do List application using Python, allowing users to add, view, update, and delete tasks efficiently.

The application stores tasks in a JSON file, ensuring data persistence even after the program is closed. Each task includes a unique identifier, a name, and a completion status, enabling structured task management. The program also includes input validation and error handling to maintain stability during execution.

This project demonstrates core programming concepts such as file handling, data structures (lists and dictionaries), control flow, and modular function design. It is designed to be simple, scalable, and easy to extend into more advanced versions, such as a graphical user interface or a web-based application.
